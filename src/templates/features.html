Responsive

Designed mobile-first. Use it on your phone, tablet or desktop.

Transparent

Your data always stays with you (as a private Gist). Whenever you make a change, your browser talks to the GitHub API directly. There is nothing else in between.

Offline

You don't have to be connected to the Internet to update your streaks. Everything is kept locally, just in case.

Hackable

strk.io provides a simple REST API that can be used to integrate with other services (see Hubot package for an example).

Open Source

strk.io is fully open source from the day one. Released under the MIT License. Available at GitHub.
