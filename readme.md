# strkio.github.io

[strk.io](http://strk.io/).

## Running locally

Make sure you have [Node.js](http://nodejs.org/) >= 0.10.

```sh
$ git clone https://github.com/strkio/strkio.github.io
$ cd strkio.github.io
$ npm install
$ bower install
$ gulp serve
```

## Deploying to GitHub Pages

```sh
$ gulp build
$ gulp deploy
```
